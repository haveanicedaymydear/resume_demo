# Trust-Aware IoV Access Control Prototype

> **Canonical standalone repository:** [haveanicedaymydear/Trust-Aware-IoV-Access-Control](https://github.com/haveanicedaymydear/Trust-Aware-IoV-Access-Control)  
> This directory is retained as a historical portfolio snapshot. Future review should use the standalone repository.

**Selected public evidence from a 2025–2026 Internet of Things coursework project.**

This project explored whether a permissioned blockchain could support distributed vehicle authentication and trust-aware access control without relying on a single central authorisation service.

The standalone repository now contains the reviewed public summary, architecture, networking evidence boundary, privacy policy, and explicit limitations. The original report, raw screenshots, interactive Packet Tracer files, complete chaincode, and some environment assets remain private where redistribution would be inappropriate.
