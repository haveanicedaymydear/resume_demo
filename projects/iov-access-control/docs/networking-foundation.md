# Supporting Networking Foundation

The IoV prototype was supported by Cisco Packet Tracer coursework and practical networking exercises.

## Retained private activity files

The surviving local evidence includes these Packet Tracer activities:

- `i408837v1n1_Packet_Tracer_Quiz.pka`
- `创建一个简单的网络.pka` (`Create a Simple Network`)
- `在 Packet Tracer 中修改和监控环境控制.pka` (`Modify and Monitor Environmental Controls in Packet Tracer`)

These activities support familiarity with:

- basic network topology construction;
- device addressing and connectivity checks;
- packet-level simulation and monitoring;
- IoT/environment-control components within Packet Tracer;
- practical troubleshooting of networked systems.

## Why the `.pka` files are not published

Cisco Packet Tracer activity files can contain course-authored content, assessment logic, and third-party assets. The original `.pka` files are therefore retained privately rather than redistributed through this public repository.

This page records the evidence boundary without claiming that the networking exercises themselves were original research contributions.
