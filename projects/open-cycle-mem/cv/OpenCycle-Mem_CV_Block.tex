\daterow{\textbf{OpenCycle-Mem: Reversible Evidence Convergence for Long-Horizon Agents}}{Jul 2026 -- Present}
\textit{Independent open-source research in progress.}
\begin{itemize}
  \item Developed a persistent agent-memory prototype with evidence-weighted checkpoint activation, provenance and source-trust gates, temporal decay, and hysteresis-based reopening or revocation under contradictory evidence.
  \item Implemented an OpenAI-compatible structured-output backend, four memory conditions, a 30-task temporal pilot suite, and 11 automated tests for auditable multi-round evaluation.
  \item Across 1,000 controlled simulation episodes, the reversible-governed method achieved 0.950 mean state accuracy versus 0.800 for static governance and 0.500 for raw memory, while reducing false-active memory to 0.050 and reopening obsolete checkpoints within 0.51 cycles on average.
\end{itemize}
